# shua's config files

feel free to use these as you'd like.

do note you will have to add `startup/startup.sh` to your ~/.bashrc.

you might also need to run `pip install -r startup/requirements.txt`.

# tty-clock
tty-clock -s -c -C 3

# pipes.sh
